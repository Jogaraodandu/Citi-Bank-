Project Overview
This project focuses on analyzing customer support tickets for a banking system to uncover trends, improve service efficiency, and enhance customer satisfaction. The dashboard provides a comprehensive view of ticket volume, resolution time, issue categories, and agent performance.

 Objectives:-
->Analyze ticket inflow and resolution trends
->Identify high-frequency issue categories
->Monitor SLA compliance and resolution time
->Evaluate support team performance
->Enable data-driven decisions to improve customer service

Tools & Technologies:-
Data Visualization: Power BI / Excel Dashboard (update if needed)
Data Processing: Microsoft Excel
Data Cleaning: Power Query / Excel
Version Control: Git & GitHub

The dataset includes:
Ticket ID
Issue Category / Type
Priority Level
Ticket Status (Open, Closed, Pending)
Created & Resolved Dates
Resolution Time
Assigned Agent / Team.

Key Metrics & KPIs:-
Total Tickets Raised
Open vs Closed Tickets
Average Resolution Time
SLA Compliance %
Ticket Volume by Category
Agent-wise Performance
Priority Distribution

Dashboard Features:
1.Dynamic filtering (Date, Category, Priority, Agent)
2.Ticket trend analysis over time
3.Category-wise issue breakdown
4.SLA tracking and delays identification
5.Performance insights for support teams

Key Insights:
->Identified peak periods with high ticket volume
->Highlighted categories causing maximum customer issues
->Detected delays in high-priority ticket resolution
->Provided visibility into agent productivity and workload distribution
